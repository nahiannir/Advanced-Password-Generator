<!-- ===================== ABOUT ===================== -->
### 🧠 About

This project is a secure password generation and management tool designed with real-world usability in mind.

Unlike basic generators, it focuses on:

- 🔐 Strong cryptographic randomness
- 💾 Safe encrypted storage
- 📋 Fast clipboard workflow
- 💪 Instant strength analysis

---


<!-- ===================== FEATURES ===================== -->
### ✨ Features

- Feature | Description

🔐 **Secure Generator:** Uses "secrets" for cryptographically strong passwords

🧠 **Memorable Passwords:** Human-friendly passwords like "ShadowNova42!"

💪 **Strength Checker:** Regex-based complexity analysis

📋 **Clipboard Copy:** Instantly copies passwords

💾 **Encryption:** Fernet-based secure storage

🔁 **CLI Interface:** Smooth, interactive experience

<!-- ===================== FUNCTIONS ===================== -->
### 🔑 Core Functions


    generate_password()

🔹 Creates a strong password using multiple character sets and ensures randomness.

    generate_memorable()

🔹 Generates readable passwords:

ShadowNova42!
TigerStorm7#

    check_strength(password)

🔹 Analyzes:

- Length
- Uppercase / Lowercase
- Numbers
- Symbols
- Returns strength level.

      save_password(password)

🔹 Encrypts using Fernet and stores safely.

    load_or_create_key()

🔹 Handles encryption key securely across sessions.
